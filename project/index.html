jChart - Real Time Organizational Chart Builder

jChart: jChart is a complete script for building any type of organizational chart in real time. As it is purely written in jQuery with 10% or less server side script, it can be used with any framework or server side language.

Script can allow you to

  • Create any level dynamic charts in real time.
  • Save charts information in database along with each node and connection info
  • Load dynamically generate charts back to original view
  • and more.
