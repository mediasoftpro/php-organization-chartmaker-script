Getting Started

jChart is a professionally written, highly customizable jquery application for building real time organizational chart or any type of charts that involve data flow or diagram demonstration.

It can allow you to create unlimited nodes and attach information with each node in real time.

It's a multi directional chart application which allow you to create node from another node in any direction. e.g top, bottom, left, right

It's a complete and easily customizable application if you have atleast working knowledge in jQuery..

jChart script is fully compatible with latest bootstrap 3.3.5. You can replace existing theme with any theme compatible with bootstrap.

Its a real-time application with 80% code written in jQuery. In backened by default used php, however you can use any server side language including asp.net, mvc frameworks, ruby on rail, php frameworks or wordpress
